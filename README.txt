The fairest coin on Monad Testnet

How to mine
-----------

1. Download foundry installer

$ curl -L https://foundry.paradigm.xyz | bash

2. Install foundry

$ foundryup

3. Get your testnet private key from your Monad Wallet

4. Run the autominer.

$ PRIVATE_KEY=MY_MONAD_TESTNET_PRIVATE_KEY ./automine.sh
